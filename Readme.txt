1. Скачать (https://aka.ms/ssmsfullsetup) и все файлы из директории Setup и установить .exe файлы.
2. Далее открыть MS SQL Manager и востоновить базу данных путем->
2.1. Нажать правой кнопкой мыши на "База данных" - выбрать"Востоновить базу данных"
2.2. Выбрать устройства - добавить - папка где храниться файл "Filmography.bak". Далее база данных должна у вас появиться в базе.
3. Теперь можно пользоваться запустить .exe файл для просмотра работы приложения (находиться в "ваша папка\Filmography\Filmography\bin\Debug".
